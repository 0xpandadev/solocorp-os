### 1人の人間が「4つのAI軍団」を指揮して、1人で企業を経営するためのAIオペレーティングシステム

> *「あなたはもうプロンプトエンジニアではない。AI軍団を指揮する1人企業のCEOだ。」*

---

## これは何か？ (What is this?)

**SoloCorp OS** は、AIに単なる「チャット相手」ではなく「役職（CEO, CTO, CMOなど）」を与え、1人の人間が数十人規模の企業のように動けるようにするための**仮想開発組織テンプレート**です。

これは単なるプロンプト集ではありません。
企画・設計・実装・品質保証・リリースという**開発工程全体を、AIに自律的に回させるための「OS（基本ソフト）」**です。人間の中途半端な介入（マイクロマネジメント）を排除し、AIがAIをレビューする「Agent-under-Agent」の構造を持っています。

---

## 4つの専門AI軍団（スキルの詳細）

SoloCorp OSには、あなたの業務を代行する4つの「軍団（エージェントの集合体）」がデフォルトでインストールされています。各軍団は内部で複数の役職に分かれ、人間の介入なしに高度な自律ループ（議論・レビュー・修正）を回します。

### 📈 1. 投資軍団 (Investment Legion)
**目的:** 投資判断、市場分析、財務評価
**内部構造:**
- **リサーチャー:** 対象企業の財務データや市場トレンドを収集。
- **クオンツアナリスト:** 収集データから「強気（Bull）」と「弱気（Bear）」の両面のシナリオを構築。
- **悪魔の代弁者（シニアパートナー）:** クオンツの分析に対して「その前提は甘い」と徹底的に批判し、リスクの抜け漏れを潰す。
- **👉 結果:** 批判に耐え抜いた、精度の高い「投資判断メモ」のみがCEOに提出される。

### 🏢 2. コンサル軍団 (Consulting Legion)
**目的:** クライアントワーク、戦略立案、課題解決提案
**内部構造:**
- **ストラテジスト:** CEOから渡された「表面的な課題」を疑い、真のボトルネック（5 Whys）を見つけ出す。
- **コンサルタント:** 課題に対する解決策を「保守的・大胆・最速」の3オプションで論理的に設計。
- **毒舌パートナー:** 「これは誰でも言える当たり障りのない提案（スロップ）だ」と厳しくダメ出しを行い、エッジの効いた提案に磨き上げる。

### 🎨 3. クリエイター軍団 (Creator Legion)
**目的:** SNSコンテンツ作成、記事執筆、マーケティング
**内部構造:**
- **トレンドアナリスト:** 現在のバズワードやターゲットの「怒り・欲求」を特定。
- **コピーライター:** 読者の目を引くフック（冒頭文）を複数作成し、初稿を執筆。
- **毒舌編集長:** 「AIっぽさ（AI Slop）」を検知し、最低2回の書き直し（リライト）を命じて、人間が書いたかのようなバイラルコンテンツに仕上げる。

### 💻 4. エンジニア軍団 (Dev Legion)
**目的:** アプリ開発、サイト制作、GitHub運用、バグ修正
**内部構造:**
- **EM（エンジニアリングマネージャー）:** アーキテクチャを設計し、最悪の障害シナリオ（Failure Mode）を事前に想定。
- **シニアコーダー:** EMの設計に基づき、堅牢なコードを実装。
- **QA / デバッガー:** 実装されたコードに対して「正常系・異常系・境界値」のテストを行い、エラーが出たら人間に報告せず自律的にコーダーに差し戻す。「絶対に動く」状態になるまで無限ループする。

## 使い方（30秒で開始）

### Step 1: このリポジトリをダウンロード（Clone）する
ターミナルを開き、自分のプロジェクトフォルダの中で以下を実行します。

```bash
git clone https://github.com/0xpandadev/solocorp-os.git
```

### Step 2: AIアシスタント（Cursor, Claude Code, OpenClaw 等）を立ち上げる
このフォルダの中でAIアシスタントを起動します。AIは自動的に `SOLOCORP.md` を読み込み、「ハブ・マネージャー」として覚醒します。

### Step 3: チャットで一言話しかけるだけ
以下のように軍団名と目的を伝えるだけで、AIが自律的に動き出します。

- 「クリエイター軍団を使って、今のAIブームでバズる記事を書いて」
- 「投資軍団を使って、〇〇社の投資可否を判断して」
- 「エンジニア軍団を使って、このディレクトリのバグを直して」

**それだけです。あとはAI軍団（内部エージェント）が全てやります。**

## 記憶と学習のシステム（Context Preservation）
あなたの指示や事業の方向性は、`notes/` ディレクトリに自動的に蓄積されます。
- `notes/INTENT.md` にあなたの「ビジョン・やりたいこと（Why）」を記載しておけば、AIは全てのタスクをそのビジョンに沿って実行します。
