# SoloCorp OS 🏢
### 1人の人間が「4つのAI軍団」を指揮して、1人で企業を経営するためのオープンソースOS

---

> *「あなたはもうプロンプトエンジニアではない。AI軍団を指揮する1人企業のCEOだ。」*

---

## これは何か？

**SoloCorp OS** は、AIに「役職」を与え、1人の人間が数十人規模の企業のように動ける「仮想開発組織テンプレート」です。

単なるプロンプト集ではありません。
これは、企画・設計・実装・品質保証・リリースという**開発工程全体をAIに自律的に回させる「OS（オペレーティングシステム）」**です。

---

## なぜ作ったか（Why）

コードを1行も書いたことのない私が、AI軍団を指揮してGitHubのトレンドに乗れることを証明したかった。

AI時代の競争優位は、**モデル単体の賢さ**ではなく、
- **仕事をどう分解するか**
- **どう役割を切るか**
- **どう工程をつなぐか**

で決まる。SoloCorp OSはその答えの1つだ。

---

## 4つの専門AI軍団

| 軍団 | 役割 | 内部の動き |
|---|---|---|
| 📈 **投資軍団** | 投資判断・財務分析 | リサーチ → クオンツ分析 → 悪魔の代弁者による自己批判 |
| 🏢 **コンサル軍団** | 戦略立案・提案書作成 | リサーチ → 戦略設計 → 毒舌レビュアーによるブラッシュアップ |
| 🎨 **クリエイター軍団** | 記事・SNS・コンテンツ作成 | トレンド分析 → 執筆 → 毒舌編集長による3回の叩き直し |
| 💻 **エンジニア軍団** | 開発・実装・QA | 設計 → コーディング → 自動テスト → バグ修正（全自動） |

---

## 使い方（30秒で開始）

### Step 1: このリポジトリをcloneする
```bash
git clone https://github.com/your-name/solocorp-os.git
```

### Step 2: あなたのAIアシスタントのワークスペースに配置する
```bash
cp -r solocorp-os/. your-project/
```

### Step 3: チャットで一言話しかけるだけ
```
「クリエイター軍団を使って、今のAIブームでバズる記事を書いて」
「投資軍団を使って、〇〇社の投資可否を判断して」
「エンジニア軍団を使って、このバグを直して」
```

**それだけです。あとはAIがやります。**
