## 这是什么？ (What is this?)

**SoloCorp OS** 是一个**虚拟开发组织模板**。它赋予AI特定的“职位”（如CEO、CTO、CMO等），让一个人能够发挥出数十人企业的能量。

它不仅仅是一个提示词集合。
它是一个**“操作系统”**，强制AI自主完成整个工程生命周期：规划、设计、实施、QA和发布。它消除了人类半吊子的微观管理，利用“Agent-under-Agent（智能体嵌套智能体）”的架构，让AI自己审核AI。

---

## 4大专业AI军团（技能库详解）

SoloCorp OS 默认安装了 4 个代表你行动的“军团（智能体集群）”。每个军团内部又划分为具体角色，在无需人类干预的情况下，运行高度自主的循环（讨论、审查、修正）。

### 📈 1. 投资军团 (Investment Legion)
**目的:** 投资决策、市场分析、财务建模。
**内部结构:**
- **研究员:** 收集目标公司的财务数据和市场趋势。
- **量化分析师:** 基于数据合乎逻辑地构建“看涨（Bull）”和“看跌（Bear）”双面场景。
- **魔鬼代言人（高级合伙人）:** 无情地批判量化分析师的结论（“你的前提太乐观了”），以消除盲点和风险。
- **👉 结果:** 只有经受住极限压力测试、高度准确的“投资备忘录”才会提交给CEO。

### 🏢 2. 咨询军团 (Consulting Legion)
**目的:** 客户服务、战略规划、问题解决提案。
**内部结构:**
- **战略家:** 质疑CEO提出的“表面问题”，并找出真正的瓶颈（使用 5 Whys 法）。
- **顾问:** 从逻辑上设计3个选项的解决方案：保守、大胆和最快。
- **毒舌合伙人:** 拒绝平庸、乏味的提案（“AI废话”），将其打磨成客户真正愿意买单的、锐利而有深度的策略。

### 🎨 3. 创作者军团 (Creator Legion)
**目的:** 社交媒体内容、文章撰写、营销。
**内部结构:**
- **趋势分析师:** 识别当前的流行语以及目标受众真正的“渴望与愤怒”。
- **文案策划:** 起草内容，生成多个“钩子”（前35个字符），并选择最强的一个。
- **毒舌主编:** 检测“AI废话（AI Slop）”，强制至少重写两次，以确保内容具有病毒传播力、独特性且充满“人味”。

### 💻 4. 工程师军团 (Dev Legion)
**目的:** App开发、网站制作、GitHub管理、Bug修复。
**内部结构:**
- **EM（工程经理）:** 在编码前设计架构，并预估最坏的故障场景（Failure Mode）。
- **高级程序员:** 根据EM的架构编写健壮的代码。
- **QA / 调试员:** 带着“搞破坏”的意图测试代码（正常路径、边缘情况、边界值）。一旦发现Bug，它会自动强制程序员修复，而不会去烦CEO。它会无限循环，直到代码“保证能用”。

## 如何使用（30秒启动）

### Step 1: 克隆此仓库 (Clone)
打开你的终端，在你的项目文件夹中运行：
```bash
git clone https://github.com/0xpandadev/solocorp-os.git
```

### Step 2: 启动你的AI助手
在此文件夹中打开你常用的AI编码助手（如 Cursor, Claude Code, OpenClaw 等）。AI会自动读取 `SOLOCORP.md`，并觉醒为你的“枢纽经理（Hub Manager）”。

### Step 3: 只需对它说一句话
简单地提及一个军团名称和你的目标即可：
- *"使用创作者军团，写一篇关于当前AI热潮的爆款文章。"*
- *"使用投资军团，判断我是否应该投资X公司。"*
- *"使用工程师军团，修复这个目录下的bug。"*

**就这么简单。AI军团（内部智能体）会处理剩下的一切。**

## 记忆保存系统 (Context Preservation)
你的所有指令和业务方向都会自动存储在 `notes/` 目录中。
- 只需将你的核心愿景写在 `notes/INTENT.md` 中，AI就会在执行所有任务时严格遵守这个“Why（初心）”。
